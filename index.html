FAQ

Häufige Fragen

Ist MC Toolkit kostenlos nutzbar? ▼
Ja, vollständig kostenlos und ohne Account. MC Toolkit läuft auf einem Server – einfach die Seite aufrufen und direkt loslegen. Keine Installation, kein lokaler Server, keine Registrierung nötig.
Wo werden meine Daten gespeichert? ▼
Nirgends. MC Toolkit läuft vollständig im Browser. Es gibt keinen Backend-Server, keine Datenbank und kein Tracking. Deine Modliste existiert nur im aktuellen Browser-Tab.
Welche Minecraft-Versionen werden unterstützt? ▼
Der Builder unterstützt 1.21.11, 1.21.4, 1.21.1, 1.20.4 und 1.20.1. Im Pack Reparatur Tool wird die Version automatisch aus der hochgeladenen .mrpack ausgelesen und kann manuell angepasst werden.
Was ist eine .mrpack-Datei? ▼
Das offizielle Modpack-Format von Modrinth. Es enthält eine Liste aller Mods mit Download-Links und Prüfsummen, aber nicht die Mod-Dateien selbst. Der Modrinth Launcher lädt die Mods beim Installieren automatisch herunter.
Kann ich auch Forge/NeoForge Mods hinzufügen? ▼
MC Toolkit ist auf Fabric ausgelegt. Forge/NeoForge-Mods können zwar gefunden werden, sind aber meist nicht kompatibel. Der Builder sucht bevorzugt nach Fabric-kompatiblen Versionen.
Ein Mod wurde als "nicht gefunden" markiert – was tun? ▼
Der Mod existiert möglicherweise noch nicht für die gewählte Minecraft-Version, oder der Slug weicht vom erwarteten Namen ab. Prüfe den korrekten Slug auf modrinth.com und trage ihn manuell über das Suchfeld ein.
Anleitung

Pack importieren

So installierst du dein fertiges Modpack in Modrinth oder CurseForge – Schritt für Schritt.

1

Pack herunterladen

Klicke in MC Toolkit auf „⬇ Pack generieren & downloaden". Die .mrpack Datei landet in deinem Download-Ordner.

💾
Downloads-Ordner → deinPack.mrpack
2

Modrinth App öffnen

Öffne die Modrinth App. Falls noch nicht installiert: modrinth.com/app → kostenlos herunterladen.

3

„Import" auswählen

Klicke oben links auf das + Symbol oder gehe zu Modpacks → rechts oben auf „Import".

📂
Modrinth App → + → Import from file
4

Datei auswählen & fertig!

Wähle deine .mrpack Datei aus dem Download-Ordner. Modrinth lädt alle Mods automatisch herunter. Danach auf „Spielen" klicken – fertig! 🎉

✅ Mods werden automatisch installiert
1

Pack herunterladen

Klicke in MC Toolkit auf „⬇ Pack generieren & downloaden" (CurseForge gewählt). Die .zip Datei landet in deinem Download-Ordner.

2

CurseForge App öffnen

Öffne die CurseForge App. Falls noch nicht installiert: curseforge.com → kostenlos herunterladen. Wähle Minecraft aus.

3

„Create Custom Profile" & Import

Klicke oben rechts auf „Create Custom Profile". Im Dialog oben auf „Import" klicken und deine .zip Datei auswählen.

📁
Create Custom Profile → Import → .zip auswählen
4

Mods aus modlist.html laden

Da die Mods von Modrinth kommen, öffne die modlist.html im ZIP. Lade alle Mods herunter und lege sie in den mods/ Ordner des Profils.

⚠ Einmalig manuell – danach automatisch
1

Pack herunterladen

Lade deine .mrpack in MC Toolkit herunter (Modrinth Format wählen).

2

Prism / MultiMC öffnen

Öffne Prism Launcher oder MultiMC. Download: prismlauncher.org

3

Neue Instanz per Import

Klicke auf „Add Instance" → links auf „Import" → „Browse" → deine .mrpack Datei auswählen → OK.

🔷
Add Instance → Import → Browse → .mrpack
4

Spielen!

Prism lädt alle Mods automatisch herunter. Doppelklick auf die neue Instanz → fertig! 🎉
